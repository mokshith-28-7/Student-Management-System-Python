# Student Management System

A simple Student Management System built using Python and Tkinter.

## Features
- Admin login system
- Add student records
- Delete student records
- Search student details
- View students in a table
- Data stored using JSON

## Technologies Used
- Python
- Tkinter (GUI)
- JSON
- PyInstaller (for creating executable)

## How to Run

1. Install Python
2. Download the project
3. Run the file:

python final_student_system.py

Default Login Password:
admin123
